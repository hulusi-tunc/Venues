# Coverage : Venues

> Browser support, device compatibility, and accessibility targets.

### Support Levels

| Level | Definition |
|-------|-----------|
| A | Full support : all features work as specified |
| B | Acceptable degradation : minor visual differences, no functional loss |
| C | Not supported : may not work, no bug fixes |

### Desktop Browsers

| Browser | Version | Windows | macOS | Level |
|---------|---------|---------|-------|-------|
| Chrome | Last 2 | A | A | A |
| Firefox | Last 2 | A | A | A |
| Safari | Last 2 | - | A | A |
| Edge | Last 2 | A | A | A |
| Opera | Last 2 | B | B | B |

### Accessibility

| Area | Target |
|------|--------|
| WCAG level | 2.1 AA |
| Keyboard navigation | All interactive elements reachable |
| Screen reader | Semantic HTML, ARIA labels on dynamic content |
| Color contrast | 4.5:1 (AA standard) |
| Zoom | Functional up to 200% zoom |
| Focus indicators | Visible on all focusable elements |
